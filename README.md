# 로컬 AI 하네스 (Local AI Harness)

윈도우·맥에서 동작하는 데스크톱 앱. 시스템 메모리를 감지해 **3단계 난이도**를 정하고, 단계에 맞는 로컬 LLM을 설치·구성한 뒤, 프롬프트 하나를 **여러 AI 에이전트가 분해·작업·검토·종합**해 결과물을 만들어 줍니다.

## 동작 원리

```
시스템 메모리 감지 → 단계 결정 → 단계별 모델 설치 → 하네스 실행
                                                  ↓
                       ① 분해 → ② 작업(병렬) → ③ 검토 → ④ 종합 → 결과물
```

### 메모리 단계

| 단계 | 총 RAM 기준 | 모델 | 실행 방식 |
|------|------------|------|-----------|
| 1 · 절약 | < 12GB | 3~4B 소형 | 단일 에이전트, 순차 |
| 2 · 균형 | 12~32GB | 7~14B 중형 | 다중 에이전트 |
| 3 · 고성능 | ≥ 32GB | 14~70B 대형 | 다중 에이전트 병렬 |

단계는 자동 감지되며, UI에서 수동으로 바꿀 수도 있습니다.

## 기술 스택

- **Tauri 2** (Rust 백엔드 + 웹뷰 UI) — 가벼운 크로스플랫폼 번들
- **Ollama** — 로컬 모델 다운로드·실행 (`localhost:11434`)
- 프런트엔드: Vanilla TypeScript + Vite

핵심 로직(메모리 감지·Ollama 연동·하네스 오케스트레이션)은 `src-tauri/src/lib.rs`에 있습니다.

## 사전 준비

1. [Ollama](https://ollama.com/download) 설치 후 실행 (`ollama serve`)
2. Rust 툴체인(`rustup`), Node.js

> 참고: 이 프로젝트는 `rust-toolchain.toml`로 Rust 1.88을 고정하고, `Cargo.lock`에서 `time` 크레이트를 0.3.47로 고정합니다. (특정 컴파일러/크레이트 조합에서 `time` 0.3.48이 빌드되지 않는 문제 회피. `cargo update`로 되돌리지 마세요.)

## 실행

```bash
npm install
npm run tauri dev     # 개발 모드
npm run tauri build   # 배포용 앱 번들 생성
```

## 하네스 알고리즘 (`run_harness`)

0. **기억 주입** — 저장된 이전 대화 요약을 전부 불러와 맥락으로 추가
1. **분해** — 플래너가 요청을 2~4개 하위 작업으로 나눔
2. **작업** — 각 하위 작업을 워커가 **병렬** 수행 (결과는 디스크 캐시로 재사용)
3. **검토** — 리뷰어가 결과의 오류·누락·모순을 점검
4. **종합** — 합성기가 하나의 답변으로 통합 (**토큰 단위 스트리밍**)
5. **요약 저장** — 이번 대화를 1~2문장으로 요약해 기억에 저장

각 단계 진행 상황은 Tauri 이벤트(`harness-step`)로, 종합 결과는 `final-token` 이벤트로 UI에 실시간 표시됩니다.

### 역할별 모델

분해 / 작업 / 검토 / 종합 단계마다 다른 모델을 지정할 수 있습니다 (UI 드롭다운). 예: 분해·검토는 빠른 소형, 종합은 큰 모델.

## 디자인 — 이미지

- **이미지 생성**: 로컬 Stable Diffusion 엔진을 자동 감지(Automatic1111 `:7860` / ComfyUI `:8188`)해 txt2img 호출. 엔진은 별도 설치·실행 필요. 결과는 `<앱 데이터>/images/`에 저장.
- **이미지 이해(비전)**: Ollama 비전 모델(`llava`, `llama3.2-vision` 등)로 이미지 분석. 이미지 첨부 + 질문 → 설명/평가.
- 둘 다 **다운로드/엔진 실행만 로컬에서 되면 오프라인** 동작.

### 기억 & 캐시 저장 위치

- 대화 요약: `<앱 데이터>/conversations.json`
- 작업 캐시: `<앱 데이터>/cache/` (모델+프롬프트 해시별)
- macOS 기준 `~/Library/Application Support/com.hongwoo.local-ai-harness/`
- UI의 **기억 초기화** 버튼으로 전부 삭제 가능
